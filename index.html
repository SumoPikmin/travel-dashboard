Travel Map

Mark countries as neutral, been or want to go. Click countries on the map to cycle the status or use the search box below.

Neutral (white)
Been (light blue)
Want to go (warm yellow)
Status is saved locally in your browser using localStorage. Click a country on the map to cycle states: Neutral → Been → Want → Neutral.
Quick actions
  • Hover countries to see their name.
  • Click to cycle status.
  • Search + Apply to set a country status directly.
Tip: use zoom (mouse wheel) and drag to pan
